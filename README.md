## 📦 Installation

```bash
# Clone repo
git clone https://github.com/AbdulVakeel/laravel-inertia-vuejs-with-tailwind-css-starter-kit.git
cd laravel-inertia-vuejs-with-tailwind-css-starter-kit

# Install backend
composer install

# Install frontend
npm install

# Setup env
cp .env.example .env

# Configure DB
# DB_DATABASE=your_db
# DB_USERNAME=root
# DB_PASSWORD=

# Generate key
php artisan key:generate

# Migrate
php artisan migrate --seed

# Run project
npm run dev
php artisan serve

# Register Admin
# http://127.0.0.1:8000/register

# Clear Cache
# http://127.0.0.1:8000/clear
